# 911-Emergency-Call-Data-Analysis
Python data analysis project exploring 911 emergency call trends, peak hours, and incident patterns using real-world data
 # Objectives

Understand call frequency trends

Identify peak call hours and days

Analyze types of emergencies

Explore time-based patterns in calls

# Tools Used

Python

Pandas

Matplotlib / Seaborn

Jupyter Notebook

# Key Analysis Performed

Data cleaning and preprocessing

Feature extraction from timestamps

Call category analysis

Time-series trend analysis

Visualization of emergency patterns

# Key Insights

Certain hours of the day show significantly higher emergency call volume

Medical emergencies were the most frequent category

Weekdays had different patterns compared to weekends

# Outcome

This analysis demonstrates how data can help improve emergency response strategies and decision-making using real-world datasets.
